## 🚀 立即使用

訪問網址：`https://你的帳號名稱.github.io/youtube-influencer-analytics/`

## 📋 使用前準備

### 1. 取得 YouTube API Key

1. 前往 [Google Cloud Console](https://console.cloud.google.com/)
2. 建立新專案或選擇現有專案
3. 啟用 **YouTube Data API v3**
4. 建立憑證（API 金鑰）
5. 複製 API Key

### 2. 設定工具

1. 在工具首頁輸入你的 YouTube API Key
2. 點擊「設定 API Key」
3. 開始使用！
